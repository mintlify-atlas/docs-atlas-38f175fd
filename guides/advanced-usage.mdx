## Overview

Beyond basic automation, Windows-MCP provides advanced tools for bulk operations, system configuration, and user notifications. These tools enable sophisticated automation workflows that combine multiple operations for maximum efficiency.

**Advanced tools covered:**
- **MultiSelect**: Select multiple items simultaneously
- **MultiEdit**: Fill multiple form fields at once
- **Registry**: Read and modify Windows Registry
- **Notification**: Send Windows toast notifications
- **Combined workflows**: Orchestrating complex automation tasks

## MultiSelect Tool

Select multiple items (files, folders, checkboxes) in a single operation.

### Basic Multi-Selection

```python
# Select multiple files with Ctrl+Click
MultiSelect(
    locs=[
        [100, 200],  # First file
        [100, 250],  # Second file
        [100, 300]   # Third file
    ],
    press_ctrl=True  # Hold Ctrl during clicks
)
```

### Using Element Labels

```python
# First capture desktop state
snapshot = Snapshot(use_vision=True)

# Then select using labels from snapshot
MultiSelect(
    labels=[5, 7, 9],  # Element labels from snapshot
    press_ctrl=True
)
```

### Multiple Clicks Without Ctrl

```python
# Perform multiple independent clicks
MultiSelect(
    locs=[
        [500, 300],
        [600, 400],
        [700, 500]
    ],
    press_ctrl=False  # No Ctrl key
)
```

## Registry Tool

Read and modify Windows Registry for system configuration.

<Warning>
**Registry modifications can affect system stability.** Only modify Registry keys you understand. Always backup before making changes.
</Warning>

### Reading Registry Values

```python
# Read a specific registry value
Registry(
    mode='get',
    path='HKCU:\\Software\\MyApp',
    name='Version'
)
```

**Returns:** `Registry value [HKCU:\Software\MyApp] "Version" = 1.2.3`

### Writing Registry Values

```python
# Set a string value
Registry(
    mode='set',
    path='HKCU:\\Software\\MyApp',
    name='ApiKey',
    value='abc123xyz',
    type='String'  # Default: 'String'
)
```

### Listing Registry Contents

```python
# List all values and sub-keys
Registry(
    mode='list',
    path='HKCU:\\Software\\MyApp'
)
```

**Returns:**
```
Values:
Version : 1.2.3
ApiKey : abc123xyz
Port : 8080

Sub-Keys:
Settings
Cache
```

### Deleting Registry Entries

```python
# Delete a specific value
Registry(
    mode='delete',
    path='HKCU:\\Software\\MyApp',
    name='OldSetting'
)

# Delete entire key and all sub-keys
Registry(
    mode='delete',
    path='HKCU:\\Software\\MyApp\\OldVersion'
)
```

### Registry Path Format

Windows-MCP uses PowerShell registry path format:

**Common root keys:**
- `HKCU:\\` - HKEY_CURRENT_USER
- `HKLM:\\` - HKEY_LOCAL_MACHINE
- `HKCR:\\` - HKEY_CLASSES_ROOT
- `HKU:\\` - HKEY_USERS
- `HKCC:\\` - HKEY_CURRENT_CONFIG

**Examples:**
```python
'HKCU:\\Software\\Microsoft\\Windows\\CurrentVersion'
'HKLM:\\SOFTWARE\\Microsoft\\Windows\\CurrentVersion\\Run'
'HKCU:\\Environment'
```

## Notification Tool

Send Windows toast notifications to alert users.

### Basic Notification

```python
# Send simple notification
Notification(
    title='Task Complete',
    message='File backup has finished successfully'
)
```

### Notification in Workflows

```python
# Alert when long operation completes
Notification(
    title='Export Ready',
    message='Your data export is ready in the Downloads folder'
)

# Alert on errors
Notification(
    title='Error Occurred',
    message='Failed to connect to server. Please check your network.'
)

# Progress notifications
Notification(
    title='Processing',
    message='Processed 50 of 100 files...'
)
```

### Notification Best Practices

<Tip>
**Keep messages concise:** Notifications disappear quickly. Keep title under 40 characters, message under 100 characters.
</Tip>

<Tip>
**Use meaningful titles:** Titles should immediately convey the notification type (success, error, progress).
</Tip>

### Example 2: System Configuration Backup

```python
# Backup critical registry settings
registry_backup = []

# Read multiple registry values
keys_to_backup = [
    ('HKCU:\\Software\\MyApp', 'Version'),
    ('HKCU:\\Software\\MyApp', 'ApiKey'),
    ('HKCU:\\Software\\MyApp', 'DatabasePath')
]

for path, name in keys_to_backup:
    value = Registry(mode='get', path=path, name=name)
    registry_backup.append(value)

# Save backup to file
backup_content = '\n'.join(registry_backup)
FileSystem(
    mode='write',
    path='C:/Backups/registry_backup.txt',
    content=backup_content
)

# Notify user
Notification(
    title='Backup Complete',
    message='Registry settings backed up to C:\\Backups'
)
```

### Example 3: Batch File Processing

```python
# Find all images
images = FileSystem(
    mode='search',
    path='C:/Pictures',
    pattern='*.{jpg,png}',
    recursive=True
)

# Open image editor
App(mode='launch', name='Paint.NET')
Wait(duration=3)

# Process each image (example for 3 images)
image_paths = [
    'C:/Pictures/img1.jpg',
    'C:/Pictures/img2.jpg',
    'C:/Pictures/img3.jpg'
]

for i, img_path in enumerate(image_paths):
    # Open image
    Shortcut('ctrl+o')
    Wait(duration=1)
    Type(loc=[500, 400], text=img_path, press_enter=True)
    Wait(duration=2)
    
    # Apply filter (example: resize)
    Shortcut('ctrl+r')
    Wait(duration=1)
    MultiEdit(
        locs=[
            [300, 200, '800'],   # Width
            [300, 250, '600']    # Height
        ]
    )
    Click(label=15, button='left', clicks=1)  # OK button
    Wait(duration=1)
    
    # Save
    Shortcut('ctrl+s')
    Wait(duration=1)
    
    # Progress notification
    if (i + 1) % 10 == 0:
        Notification(
            title='Processing Images',
            message=f'Processed {i + 1} of {len(image_paths)} images'
        )

# Final notification
Notification(
    title='Batch Processing Complete',
    message=f'Processed {len(image_paths)} images successfully'
)
```

## Best Practices for Advanced Workflows

<AccordionGroup>
  <Accordion title="Error handling and recovery">
    Always include error checks and fallback options:
    
    ```python
    # Attempt operation
    result = FileSystem(mode='copy', path='source.txt', destination='dest.txt')
    
    # Check for errors
    if 'Error' in result:
        # Try alternative
        FileSystem(mode='copy', path='backup.txt', destination='dest.txt')
        
        # Notify user
        Notification(
            title='Warning',
            message='Used backup file instead of original'
        )
    ```
  </Accordion>

  <Accordion title="Progress tracking">
    Use notifications for long-running operations:
    
    ```python
    total_items = 100
    
    for i in range(total_items):
        # Process item...
        
        # Send progress notification every 25%
        if (i + 1) % 25 == 0:
            Notification(
                title='Progress Update',
                message=f'{i + 1}/{total_items} items processed'
            )
    ```
  </Accordion>

  <Accordion title="Registry safety">
    Always backup before modifying registry:
    
    ```python
    # Backup current value
    old_value = Registry(
        mode='get',
        path='HKCU:\\Software\\MyApp',
        name='Setting'
    )
    
    # Save backup
    FileSystem(
        mode='write',
        path='registry_backup.txt',
        content=old_value,
        append=True
    )
    
    # Apply new value
    Registry(
        mode='set',
        path='HKCU:\\Software\\MyApp',
        name='Setting',
        value='new_value'
    )
    ```
  </Accordion>

  <Accordion title="Validation before bulk operations">
    Verify before executing bulk changes:
    
    ```python
    # Preview what will be affected
    files_to_process = FileSystem(
        mode='search',
        path='C:/Data',
        pattern='*.txt'
    )
    
    # Show count to user (via notification or return value)
    # Then proceed with bulk operation
    ```
  </Accordion>
</AccordionGroup>
