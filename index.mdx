## Welcome to Windows-MCP

Windows-MCP is a lightweight, open-source MCP (Model Context Protocol) server that bridges AI agents and the Windows operating system. It enables LLMs to perform desktop automation tasks such as file navigation, application control, UI interaction, and QA testing through 15 powerful tools.

## Key Features

<CardGroup cols={2}>
  <Card title="Seamless Windows Integration" icon="windows">
    Interacts natively with Windows UI elements, opens apps, controls windows, and simulates user input
  </Card>
  <Card title="Works with Any LLM" icon="brain">
    No computer vision or fine-tuned models required - reducing complexity and setup time
  </Card>
  <Card title="15 Powerful Tools" icon="tools">
    Complete toolset for keyboard, mouse, window management, and system operations
  </Card>
  <Card title="Real-Time Performance" icon="bolt">
    0.2-0.9 second latency between actions depending on system load
  </Card>
  <Card title="DOM Mode for Browsers" icon="browser">
    Special mode for web automation that filters browser UI for cleaner interaction
  </Card>
  <Card title="Local & Remote Modes" icon="server">
    Run directly on Windows or connect to cloud-hosted Windows VMs
  </Card>

## Getting Started

<Steps>
  <Step title="Install Prerequisites">
    Ensure you have Python 3.13+ and UV package manager installed on your Windows system
  </Step>
  <Step title="Install Windows-MCP">
    Use `uvx windows-mcp` for quick PyPI installation or clone from source
  </Step>
  <Step title="Configure Your MCP Client">
    Add Windows-MCP to Claude Desktop, Perplexity, Gemini CLI, or other MCP clients
  </Step>
  <Step title="Start Automating">
    Begin using the 15 tools to automate Windows desktop tasks
  </Step>
</Steps>

## Supported Operating Systems

Windows-MCP supports all modern Windows versions:

- Windows 7
- Windows 8, 8.1
- Windows 10
- Windows 11
